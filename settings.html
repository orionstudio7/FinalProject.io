Description

As I learn more about accessible design, I want to continue to update this website to make it more accessible. As of 11/12/2023, I have created the html and css format for users to set the font of the website to OpenDyslexic. All of the examples I have been looking at, however, use javascript to change the font of the website. As of 12/9/2023 I added functionality to allow the user to change the website to OpenDyslexic using javascript. The Open Dsylexic font is a project created by Abbie Gonzalez. It has been designed around "some common symptoms of dyslexia". You can find out more here.
